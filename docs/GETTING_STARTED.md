# Getting Started

## Prerequisites

- [Git](https://git-scm.com/install)
- [Go](https://go.dev/doc/install)
- Network access to Microsoft Graph (for API calls)
- [Recommended] [`grpcurl`](https://github.com/fullstorydev/grpcurl) for testing gRPC endpoints

---

## Use the example server

1. Clone the repo

```bash
git clone https://github.com/bautistv/posta-baut.git
cd posta-baut
```

2. Run the server

```bash
go run ./cmd/main/main.go
```

See the [runbook](./RUNBOOK.md) for more detailed instructions.
